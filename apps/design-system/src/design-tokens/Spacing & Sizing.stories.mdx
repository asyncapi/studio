# Spacing & Sizing

By default the spacing scale is inherited by the `padding`, `margin`, `width`, `height`, `maxHeight`, `gap`, `inset`, `space`, and `translate` tokens.
Please note that we're using a scale of 4. E.g., `1` is 1/4 (0.25), `8` is 8/4 (2), etc.

> **Note:** If you ever need to create a new value, make sure you follow the scale.
