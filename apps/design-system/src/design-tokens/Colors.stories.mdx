# Colors

|Token|Description|Example|
|:--|:--|:--|
|`bg-{name}-{swatch}`|Background color|`bg-red-500`|
|`text-{name}-{swatch}`|Text color|`text-gray-900`|
|`border-{name}-{swatch}`|Border color|`border-pink-600`|

Along with these, you can also use special tokens like `{bg|text|border}-{transparent|white|black|primary}`. For instance, `bg-primary`, `text-white`, `border-black`, and so on.

The following is a strict list of colors supported on Studio:
