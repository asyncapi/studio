# Borders

## Border width

|Token|Description|
|:--|:--|
|`border-0`|No border|
|`border-2`|2px border|
|`border-4`|4px border|
|`border-8`|8px border|

For more variations, see https://tailwindcss.com/docs/border-width.

## Border style

|Token|Description|
|:--|:--|
|`border-solid`|(Default) Solid border|
|`border-dashed`|Dashed border|
|`border-dotted`|Dotted border|
|`border-none`|No border|

For more variations, see https://tailwindcss.com/docs/border-style.

## Border radius

|Token|Description|
|:--|:--|
|`rounded-none`|(Default) No rounded border|
|`rounded-sm`|2px radius|
|`rounded`|4px radius|
|`rounded-md`|6px radius|
|`rounded-lg`|8px radius|
|`rounded-xl`|12px radius|
|`rounded-2xl`|16px radius|
|`rounded-3xl`|24px radius|
|`rounded-full`|Circle, Ellipse|

For more variations, see https://tailwindcss.com/docs/border-radius.
